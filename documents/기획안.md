# 모아협동조합 홈페이지 구축 기획안 (v1.1)

## 1. 프로젝트 개요 (Overview)
*   **목표:** 영리 사업을 통한 수익 창출과 활발한 사회 공헌 활동(봉사)을 동시에 효과적으로 홍보하여, 일반 고객의 신뢰를 얻고 대기업(예: KT&G)의 후원(CSR)을 유치함.
*   **컨셉:** "따뜻한 전문가 (Warm Professional)"
*   **핵심 가치:** 영리 활동 50% + 사회 공헌 50%의 균형감 있는 노출.

---

## 2. 도메인 및 배포 전략 (Domain & Deployment)

### 2.1 도메인 (Approved)
*   **선정 도메인:** `moacoop.co.kr`
    *   'Coop'은 전 세계적으로 널리 쓰이는 협동조합의 약칭으로 신뢰감을 줍니다.

### 2.2 배포 및 DNS 연결 가이드
*   **호스팅:** Vercel (무료 플랜 활용)
*   **연결 목표:** 사용자가 `http://moacoop.co.kr` 접속 시 Vercel 배포 페이지(`https://moacoop.vercel.app`)로 연결.
*   **DNS 설정 방법 (가비아/호스팅케이알 등 도메인 등록 업체 설정 페이지):**
    1.  **Vercel 대시보드:** Project Settings > Domains > `moacoop.co.kr` 추가.
    2.  **도메인 관리 사이트 (DNS 설정):**
        *   **A 레코드 (필수):**
            *   호스트: `@` (또는 비워둠)
            *   값/IP: `76.76.21.21` (Vercel 고정 IP)
        *   **CNAME 레코드 (선택 - www 연결 시):**
            *   호스트: `www`
            *   값: `cname.vercel-dns.com`
    3.  **확인:** 설정 후 약 10분~1시간 내에 SSL 인증서가 자동 발급되며 접속이 가능해집니다.

## 3. 디자인 가이드라인 (Design Guidelines)

**"Simple is Best"**를 원칙으로 하며, 모바일 환경을 최우선으로 고려합니다.

### 3.1 UX/UI 원칙
*   **Mobile First:** 모든 디자인은 모바일 폭(375px~430px)을 기준으로 먼저 기획하고, PC 화면으로 확장합니다.
*   **반응형 (Responsive):** 데스크탑, 태블릿, 모바일 모든 기기에서 최적화된 화면을 제공합니다.
*   **직관적 탐색:** 햄버거 메뉴(모바일)와 명확한 GNB(PC)를 통해 2번 클릭 이내에 원하는 정보에 도달해야 합니다.

### 3.2 컬러 시스템 (Color System) - Strict Mode
지정된 4가지 색상 외에는 사용을 엄격히 금지합니다. (필요시 명도/투명도 조절만 허용)

| 구분 | 색상 코드 | 용도 | 비고 |
| :--- | :--- | :--- | :--- |
| **Main Color** | **#0056b3** (예시) | 브랜드 아이덴티티, 버튼, 강조 텍스트 | 신뢰감을 주는 짙은 블루 계열 추천 |
| **Sub Color** | **#f5a623** (예시) | 봉사/따뜻함 강조, 포인트 아이콘 | 희망을 상징하는 옐로우/오렌지 계열 |
| **Black** | **#111111** | 헤드라인, 주요 본문 | 완전한 #000보다는 아주 짙은 회색 권장 |
| **White** | **#FFFFFF** | 배경, 카드 박스 배경 | 깔끔함 유지 |

### 3.3 타이포그래피 (Typography)
*   **폰트:** 가독성이 좋은 산세리프 계열 (Pretendard 또는 Noto Sans KR 추천)
*   **글자 색상:**
    *   **기본 텍스트:** `#333333` (가독성 확보)
    *   **보조 텍스트:** `#666666` (설명, 날짜 등)
    *   **강조/헤드라인:** `#111111` 또는 **Main Color**
    *   **배경 위 텍스트:** `#FFFFFF`

## 4. 기술 스택 및 보안 (Tech Stack & Security)

### 4.1 핵심 스택
*   **Framework:** **Next.js 15 (Latest Stable)**
    *   최근 발견된 보안 취약점(CVE-2025-66478 등)이 해결된 최신 버전을 반드시 사용합니다.
*   **Package Manager:** **pnpm**
    *   빠른 설치 속도와 디스크 공간 효율성을 위해 npm 대신 사용합니다.
*   **Styling:** **Tailwind CSS**
    *   Utility-first 방식을 통해 빠른 모바일 반응형 구현.
*   **Deployment:** **GitHub Actions (Optional) + Vercel**
    *   GitHub Push 시 Vercel 자동 배포 연동.

### 4.2 개발 환경 설정
*   Node.js: LTS 버전 (v20 이상 권장)
*   `pnpm` 설치 및 초기화 필수.
